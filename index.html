Rules
Basic Information
    Lives:
  • Player has 3 lives.
  • Each time player have a collision with the enemy it looses 1 life.
  • When there are 0 lives left the game ends with the final screen.
    Points:
  • Each time player reaches the water +100 points counted.
  • Number of points at the end of the game is the player result.
